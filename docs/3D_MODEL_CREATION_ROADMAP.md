## üìù Model Specifications

### File Format
- **Primary:** GLB (GLTF binary) - optimized for web
- **Source:** Blender, 3DS Max, Maya, or similar
- **Texture Maps:** Diffuse, Normal, Roughness, Metallic
- **Target Polycount:** 5K-50K triangles (depends on complexity)

### Technical Requirements
- **Scale:** Objects should be appropriately sized relative to each other
- **Pivot Points:** Center at origin for easier positioning
- **Materials:** PBR (Physically Based Rendering) materials preferred
- **Optimization:** Bake shadows and lighting where possible

### Integration Steps
1. Export as GLB from Blender/3D software
2. Add to `public/models/` directory
3. Update `modelRegistry.ts` with new model entry
4. Add keyword matching to shopping list items
5. Test in 3D viewer with actual shopping lists
